Running the Test
---------------- 
**************************************************


Dependencies:

The Trader Server must be running.
To start the Server:

Unix:  	run the "runtrader.sh" script


The server scripts are located the parent directory of this directory.

All scripts are configured with ORB.PortNum and ORB.OrbName already set.  If there
are conflicts with these change the values of these variables and re-run the test.
