757     *
758     * Note: DerivedQuote has two sides: AsDefined and Opposite sides. The prices for

760     * means a credit. Let b the price for AsDefined side, and a the price for Opposite
761     * side. It is assumed that: b + a > 0. (If b + a <= 0, then the derived quote is crossed).
762     *
763     * 1. A price p for AsDefined side is said to be inside the derived quote if
764     *    -a < p < b.
765     * 2. A price p for Opposite side is said to be inside the derived quote if
766     *    -b < p < a
767     */
768    public boolean isInsideDerivedQuote(Price aPrice, Side aSide){
769        long p = aPrice.toLong();
770        boolean result = false;
771        Price price = getBid().getPrice();
772        long bid = 0;
773        if (!price.isNoPrice())
774        {
